# Encurtador de Links

Projeto feito em Java com Spring Boot para encurtar URLs.

## Tecnologias
- Java 21
- Spring Boot
- Spring Web
- Spring Data JPA
- H2 Database
- Thymeleaf

## Funcionalidades
- Encurtar links
- Redirecionar URLs
- Interface web simples

## Como rodar

```bash
mvn spring-boot:run
